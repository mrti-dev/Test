Lập trình Web

Lập trình Web cung cấp các kỹ năng cơ bản và nâng cao để phát triển ứng dụng web hiện đại. Khóa học bao gồm:

  1. HTML, CSS, JavaScript: xây dụng giao diện trang web thân thiện và tối ưu
  2. Phát triển giao diện người dùng: Sử dụng framework như broadcast để tạo giao diện responsive
  3. Backend với PHP, Node.js
  4. Cơ sở dữ liệu:
    • MySQL
    • PostgreSQL
